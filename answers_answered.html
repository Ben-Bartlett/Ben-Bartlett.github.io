Lab Reflection Answers

Understanding the Basics

1. What is the purpose of <!DOCTYPE html> in an HTML file?
It tells the browser that the file is an HTML5 document.
2. What are the key structural elements in an HTML document?
They include <html>, <head>, <title>, <body>, and various content elements.
3. How do comments work in HTML, and why are they useful?
Comments are written using <!-- comment -->. They help document the code for future reference.
4. What is the purpose of the <head> and <body> sections of an HTML file?
The <head> contains metadata, links to stylesheets, and scripts. The <body> contains the main content of the page.

Working with Links

5. How do you create a hyperlink in HTML?
Using the <a> tag: <a href="url">Link Text</a>.
6. What does the target="_blank" attribute do when linking to an external website?
It opens the link in a new tab or window.
7. What happens if you link to a page that does not exist in the directory?
The browser displays a 404 error (Page Not Found).

Practical Reflection

8. What challenges did you face while creating the HTML files?
Answers will vary, but common challenges include syntax errors and incorrect file paths.
9. What is the benefit of using a structured text editor like VS Code compared to a basic one?
VS Code provides syntax highlighting, auto-completion, debugging tools, and an integrated terminal, making development easier.
